# Sicherheitsrichtlinie

## Schwachstellen melden

Bitte keine öffentlichen GitHub-Issues für Sicherheitslücken anlegen.
Meldungen gehen vertraulich per E-Mail an:

**contact@pangea-intelligence.eu**

Hilfreich sind: betroffene Version, Reproduktionsschritte oder eine
Beispiel-Datei sowie erwartetes und tatsächliches Verhalten.

## Ablauf

- Eingangsbestätigung innerhalb von 7 Tagen.
- Wir prüfen die Meldung, halten die meldende Person auf dem Laufenden
  und veröffentlichen einen Fix als neue Version.
- Bitte die Schwachstelle nicht öffentlich machen, bevor ein Fix
  verfügbar ist.

## Unterstützte Versionen

| Version          | Unterstützt |
| ---------------- | ----------- |
| neueste 0.x      | ja          |
| ältere Versionen | nein        |

Sicherheitsfixes erscheinen nur für die jeweils neueste 0.x-Version.
